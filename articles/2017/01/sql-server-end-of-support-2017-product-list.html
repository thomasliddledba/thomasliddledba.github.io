SQL Server Products Reaching End of Support in 2017

Microsoft SQL Server 2012 Service Pack 2 reached End of Support yesterday (January 10, 2017). Microsoft SQL Server 2014 Service Pack 1 will reach end of support on October 10th, 2017.

Be sure to look at your environment and upgrade as needed. The link below is Microsoft's Link to Products that are Reaching End of Support in 2017.
